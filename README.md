
# 🏮 MTU-JSON-GUI (MTU嵌字编辑器)

### 专业的日漫汉化可视化排版工作站

**MTU-JSON-GUI** 是一款专为日漫汉化器 https://github.com/hgmzhn/manga-translator-ui https://gitee.com/hgmzhn/manga-translator-ui 量身定制的 Web 端可视化嵌字工具。它不仅能处理文本替换，更内置了一套精密平衡的**几何排版引擎**，旨在通过算法自动化解决日漫排版中繁琐的对齐、间距与透视问题。

---

## 🏗️ 核心特色功能：

**MTU-JSON-GUI** 不仅是一个文本编辑器，其核心在于内置了一套精密平衡的**几何排版引擎**，旨在通过算法自动化解决繁琐的对齐、间距与透视问题。

---

## 🏗️ 1. 全场景几何排版引擎 (Typography Engine)

### 🔹 万能标点对齐算法 (Ink-Middle Alignment)
* **物理边界测量**：系统通过测量字符墨迹的真实物理边界（`actualBoundingBox`），而非依赖字库自带的排版占位。
* **几何中心对齐**：无论使用何种字体，确保 `！`、`？`、`…` 等标点符号在任何语境下都能精准对齐中轴线。
* **智能压缩与贴边**：自动识别前/后括号及句读点，根据排版方向动态执行压缩和贴边补偿，消除视觉散乱感。

### 🔹 极致的对齐控制
* **块居中 (Block-Center-Top)**：将整个文本块作为一个整体在对话框内居中，保持行间逻辑一致。
* **多样化模式**：支持块首对齐、块尾对齐以及每一列文字独立居中的“单列居中”模式。

---

## 🤖 2. 像素级自动化套件 (Automation Suite)

* **感知式自动字号 (Auto FontSize)**：通过 100px 基准预渲染测量，根据文本框的物理空间自动推导出最契合的字号。
* **动态自动行距 (Auto Line Spacing)**：自动分析 OCR 原始行块的投影距离，一键还原原版漫画的行分布比例。
* **智能自动换行 (Auto-Wrap)**：基于像素宽度的精确拆分，竖排自动分列，横排自动分行，并完美保持富文本标签的属性继承。
* **自动 TCY (纵中横)**：智能识别 2-4 位数字及 `!?` 组合并执行旋转排版，适配日式竖排习惯。

---

## 🎨 3. 深度富文本与变换引擎 (Rich Text Engine)

支持在单行文本内混用多种复杂的装饰性样式，系统会实时将其解析为绘制状态机。

| 标签 | 功能描述 | 示例与参数说明 |
| --- | --- | --- |

## 🛡️ 4. 商业级渲染系统 (Visual System)

* **强力填孔描边 (Fill-Holes System)**：采用“向心圆叠加算法”（Concentric Rendering），彻底消除传统 Canvas 描边在大字号下出现的内部空心与破碎现象。
* **智能白色探测 (Auto-White Detection)**：识别到接近白色的文字时，自动开启 0.2 倍率的黑色描边，确保在任何背景下的可读性。
* **多维视觉效果**：支持实时调节描边圆角、发光 (Glow) 模糊半径及颜色。

---

## 🧹 5. 智能修图与生产力工具 (Cleanup & Productivity)

* **自动背景填充 (Fill)**：智能检测行框边缘背景色，自动生成带圆角补偿的填充块，极大减少手动修图工作。
* **混合模式 (Combo)**：支持右键自由涂鸦，中键蒙版橡皮，无需切换工具即可完成精细修补。
* **内存感应系统**：自动检测硬件内存（`deviceMemory`），为低内存设备自动限制并发数以防浏览器崩溃。

---

# 🏮 MTU-JSON-GUI 使用指南

**MTU-JSON-GUI** 是一款专为漫画汉化设计的专业级可视化排版工作站。它通过像素级的渲染算法，实现了从 OCR 数据到最终嵌字成稿的全自动化与精细化控制。

---

## 📂 核心导入指南 (Import Guide)

### 0.下载 https://github.com/charlespfan/mtu-json-gui/releases/ 中的html文件，在联网情况下用浏览器打开，推荐vivaldi浏览器

### 1. 命名规范与智能匹配
本工具的导入系统具备智能识别逻辑，建议严格遵守以下命名规范以获得最佳体验：
为了实现原图、JSON 数据与修复图（去字图）的“秒速连连看”，请确保文件名符合以下规则：

* **原图**：`001.jpg`
* **JSON 数据**：`001_translations.json` (必须包含 `_translations` 后缀)
* **修复图**：`001_inpainted.jpg` 或 `001.png` (系统会自动去除 `_inpainted`, `_fixed`, `_mask` 等后缀进行匹配)

### 2. 导入模式选择

* **一次性批量导入（推荐）**：自动识别 MTU 标准项目结构，一次性加载所有相关资源。
* **单文件模式**：点击顶栏“单 JSON”、“单原图”和“单原图”按钮，手动配对单个页面。
* **批量文件夹模式**：
1. 点击 **“原图文件夹”**，选择包含所有原图的文件夹。
2. 点击 **“JSON文件夹”**，选择包含所有翻译数据的文件夹。
3. 点击 **“修复图文件夹”**，选择包含所有修复图（去字图）的文件夹。
4. 系统将自动执行**自然数排序**（确保 2.jpg 在 10.jpg 前面）并激活右侧分页导航栏。


## 功能手册：

### 一、 顶部工具栏 (文件与全局设置)

#### 1. 导入与文件管理
* **单 JSON / 单原图 / 单修复图**：分别导入单个对应文件。
* **JSON / 原图 / 修复图文件夹**：批量导入整个文件夹内的文件。
* **一次性批量导入（推荐）**：自动识别 MTU 标准项目结构，一次性加载所有相关资源。
* **导出单图**：将当前画布看到的成品导出为单张图片。
* **批量导出到文件夹**：直接调用浏览器文件系统 API，将所有页面渲染并保存到本地文件夹（无需打包）。
* **批量导出为压缩包**：将所有页面渲染后打包成一个 `.zip` 文件下载。
* **并发渲染数**：设置批量导出时的同时处理数量（低内存设为 1-2，有显卡设为 4+）。
* **导出当前/批量导出 JSON**：保存当前的编辑进度、坐标、排版样式等元数据。

#### 2. 视图与显示控制
* **全图修复**：开启后强制显示完整修复图，忽略蒙版数据。
* **总框 / 行框 / 文字 / 显蒙**：控制 UI 辅助线（大框、小行框）、文字内容、红色蒙版层的可见性。
* **修复层透明度**：调节 Inpaint 修复图的透明度，方便对比原图。
* **译 / 原 按钮**：
    * **译**：进入译文编辑模式，显示翻译后的文字和排版手柄。
    * **原**：进入原文校对模式，显示 OCR 识别的原文和对应的小行框手柄。

#### 3. 页面导航
* **第一页 / 末一页 / 上一页 / 下一页**：在批量导入的文件间切换。
* **页码输入框**：输入数字按回车直接跳转。

#### 4. 全局排版样式
* **字体选择**：选择内置或已加载的字体。
* **检**：自动检测并列出当前电脑系统安装的所有字体。
* **+ (字体加载)**：从本地上传 `.ttf`、`.otf` 字体文件。
* **字号 (自)**：手动设置字号，或点击“自”对当前页/框执行自动字号计算。
* **字重**：设置文字的粗细（400 为常规，700/900 为粗体）。
* **对齐方式 (↺)**：设置块居中、单列居中等。点击 `↺` 强制将当前对齐方式应用到全书。
* **标点模式 (↺)**：设置标点是占半格还是全格。点击 `↺` 全局应用。
* **行间距 / 字间距 (自)**：手动调节间距。行间距点击“自”可根据框高度自动分配。
* **纵中横 (TCY)**：自动将竖排中的短数字和 !? 组合转为横排显示。
* **颜色与描边**：
    * **字体颜色 / 描边颜色**：点击色块弹出原生取色器。
    * **描边开关 / 粗细 / 圆 / 填孔**：控制描边的开启、宽度、圆角，以及是否填充笔画空洞。
    * **发光 (Glow)**：设置文字外发光效果的开关、颜色和模糊程度。

---

### 二、 左侧边栏 (编辑与富文本)

#### 1. 富文本标签 (RT 标签)
* **[B], [C], [I], [V], [Rot]**：快速给选中文字加粗、设色、斜体、切变、旋转。
* **[Mh], [Mv]**：文字水平或垂直翻转镜像。
* **[K], [X], [Y]**：微调文字间距和局部偏移。
* **[T], [nT]**：局部强制或取消纵中横。
* **[•] (着重号)**：在文字旁添加着重号点。
* **[Sz], [Fn], [Ru]**：局部修改字号、字体或添加注音（Ruby）。
* **[NoBR]**：一键清除译文框内所有换行符。
* **[Clear]**：清除选中文字的所有富文本标签。

#### 2. 常用符号快捷键
* 包含日漫常用的全角标点（！、？、…、——）、引号（「」、『』）、旋转感叹号、以及特殊符号（♥、♪、█、全角空格）。

#### 3. 文本编辑区
* **Translation (译文)**：主要的嵌字内容输入框。
* **临时文本区**：跨页存储的草稿箱，内容会自动保存到浏览器缓存。
* **Original OCR (原文)**：显示或修改 OCR 的识别结果，用于原文模式对照。

#### 4. 底部功能
* **↺ 重置**：撤销当前框的所有文字改动，恢复到本页刚加载时的状态。
* **斜 / 直**：智能校准功能。
    * **斜**：自动分析文字倾斜度并调整外壳角度。
    * **直**：将文字角度归零，恢复水平或垂直。

---

### 三、 右侧边栏 (工具快捷按钮)

* **一 (全自动)**：一键执行“文本修正规则 + 自动字号 + 自动行距”。
* **替**：仅执行原文标点符号的修正替换规则（如将 `．．` 换为 `…`）。
* **自**：为所有页面批量执行自动字号和行距计算。
* **＋ (绘制模式)**：开启后通过鼠标拖拽直接在画布上新增一个文字区域。
* **合**：将选中的多个行框合并为一个总框。
* **拆**：将选中的行从小框中剥离，变为独立的文字块。
* **属**：切换选中框的横竖排属性。
* **删**：删除选中的文字框或行框（按 Alt+左键划线可多选删除）。
* **译文查找替换**：弹出一个对话框，支持在当前页或全书中查找和替换译文。
* **🎨**：调色盘，用于快速精准替换富文本中的十六进制颜色代码。
* **🧽 (蒙版橡皮)**：编辑蒙版，用于擦除不需要的修复部分。
* **🖌️ (自由涂鸦)**：直接在画面上绘画（可改颜色），常用于手动补色。
* **混 (混合模式)**：右键涂鸦，中键橡皮，高效修图。
* **填 (自动填充)**：智能检测行框边缘并填充纯黑或纯白背景。

---

### 四、 画布交互 (隐藏操作)

* **鼠标左键**：选中、拖拽移动。
* **鼠标右键**：
    * 在空处：清除右键多选。
    * 在框上：多选大框。
    * 在涂鸦模式下：绘画/涂色。
* **鼠标中键**：在修图模式下执行相反操作（如橡皮变画笔）。
* **Alt + 左键划线**：划线多选（穿过的行框都会被选中，用于合并、拆分或删除）。
* **Shift + 左键点击**：连续多选。
* **空格 + 左键拖拽**：平移视图。
* **滚轮**：缩放视图（以鼠标位置为中心）。
* **快捷键 `[` / `]`**：快速增减选中文字的字号。
* **快捷键 `<` / `>`**：在修图模式下调节笔触大小。
* **方向键 `←` / `→`**：快速切换上一页 / 下一页。
